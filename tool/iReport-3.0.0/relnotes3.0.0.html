iReport Release 3.0.0
May 27, 2008

The following changes have been made to software since release 2.0.5
  • Updated to JasperReports 3.0.0
  • New expression based properties for report elements
  • New hyperlink targets (parent and top)
  • Updated JasperServer plugin 3.0.0
  • Several bug fixes
Contributors this time:
  • Giulio Toffoli
Thanks for your suggestions and your on going support using iReport!

Giulio
